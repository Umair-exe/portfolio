# Professional Development Services

A modern, professional services website built with Next.js, React, Framer Motion, and Tailwind CSS. Showcasing full-stack development services, case studies, and technical expertise.

## üöÄ Features

- **Smooth Animations**: Eye-catching animations powered by Framer Motion
- **Responsive Design**: Looks great on all devices (mobile, tablet, desktop)
- **Dark Mode**: Sleek dark theme for better viewing experience
- **Interactive UI**: Hover effects, scroll animations, and smooth transitions
- **Type Animation**: Dynamic typing effect in hero section
- **Glass Morphism**: Modern glassmorphism design elements
- **Fast & Optimized**: Built with Next.js for optimal performance
- **Service-Focused**: Clear presentation of services, case studies, and expertise

## üé® Customization

### Update Personal Information

Edit the following files to customize the content:
- `components/Hero.tsx` - Update name, tagline, and contact info
- `components/About.tsx` - Modify the about section
- `components/Experience.tsx` - Add/edit work experience
- `components/Projects.tsx` - Showcase your projects
- `components/Skills.tsx` - Update your technical skills
- `components/Contact.tsx` - Modify contact information

### Update Social Links

In `components/Hero.tsx` and `components/Contact.tsx`, update:
- GitHub URL
- LinkedIn URL
- Other social media links

### Color Scheme

Colors can be customized in `tailwind.config.js`:
```javascript
colors: {
  primary: {
    // Update primary colors here
  }
}
```
